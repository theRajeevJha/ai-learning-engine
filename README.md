# ai-learning-engine
Adaptive learning engine using knowledge graphs and mastery tracking for personalized education.

## GitHub Pages

I added a simple site in the `docs/` folder and a GitHub Actions workflow to publish it automatically to the `gh-pages` branch on pushes to `main`.

- Site source: `docs/index.md`
- Deploy workflow: `.github/workflows/deploy-gh-pages.yml`

After the first successful workflow run, the site will be available at:

https://theRajeevJha.github.io/ai-learning-engine

If you prefer to serve Pages from the `main` branch `/docs` folder instead, go to the repository Settings → Pages and select "Branch: main /docs folder".
